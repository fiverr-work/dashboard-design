    Purpose of this form Use this form to apply for a Mortgage Advantage. If there are two personal applicants, both applicants need to sign the one application form. Use a separate application form for each package if you require a package for personal lending and a package for company lending. You can only establish a Mortgage Advantage in the name.

    Part A: What you need to know You cannot establish a Mortgage Advantage in conjunction with Low Documentation loans. The annual package fee is non-refundable and will be charged to the nominated home loan participating in the package (in some circumstances, for example interest-only loans, the annual fee may be redirected to a related transaction account for the eligible home loan). If at any time there is no nominated home loan, we will determine the home loan to which the package fee is to then be charged in our discretion. If you would like any information about the current nominated home loan or to change the home loan to which the fee will be charged, visit any CommBank branch or call 13 2224, 8am to 8pm. For information about the terms and conditions that apply to Mortgage Advantage please refer to Mortgage Advantage Fact Sheet (on-platform) and Package Terms and Conditions.
